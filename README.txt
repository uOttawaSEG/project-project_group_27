## Project Description
OTAMS (Online Tutoring Appointment Management System) is an Android application designed to help the University of Ottawa Help Centre manage tutoring appointments.  
It supports three roles: Student, Tutor, and Administrator.  
Deliverable 1 implements:
- Student and Tutor registration forms  
- Login and logout functionality  
- Display of the user’s role after login  
- Input validation for required fields  
- Firebase Realtime Database integration (optional)
---
## How to Use the App
1. **Install the APK:** `Project_Group_27_debug.apk`  
2. Open the app and choose **Register.**  
   - Fill in the Student or Tutor form and submit to create an account.  
3. Return to the **Login** screen and enter the email and password you used.  
4. After successful login, the screen shows:  
   “Welcome! You are logged in as <Student / Tutor / Admin>.”  
5. Click **Logout** to return to the main screen.
